# Programacion Orientada a Objetos

Este repositorio corresponde a las primeras clases de la materia **Programacion Orientada a Objetos** de la **Universidad Nacional de Jose C. Paz (UNPAZ)**.

Por el momento, el proyecto representa una base inicial de trabajo para la cursada. La idea es utilizar este espacio para practicar los conceptos vistos en clase y avanzar progresivamente con ejercicios, ejemplos y desarrollos realizados durante la materia.

## Contexto

La materia esta orientada al aprendizaje de los fundamentos de la programacion orientada a objetos utilizando Java, abordando temas como clases, objetos, atributos, metodos, encapsulamiento y relacion entre entidades.

Este proyecto todavia se encuentra en una etapa inicial, correspondiente a las primeras practicas de la cursada.

## Objetivo

Contar con un proyecto base sobre el cual ir incorporando, clase a clase, los contenidos y ejercicios trabajados en la materia.
